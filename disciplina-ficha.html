Objectivos da Disciplina

PT
  1. Identificar ambientes de execução e ferramentas usadas no desenvolvimento de software.
  2. Identificar características das linguagens de programação.
  3. Desenvolver programas usando Java (ou outra).
  4. Conhecer e implementar principais tipos de dados.
  5. Desenvolver e implementar algoritmos.
EN
  1. Identify different execution environments and tools used in software development.
  2. Identify main characteristics of programming languages.
  3. Develop programs using Java (or other).
  4. Know and implement main data types.
  5. Develop and implement an algorithm.
